# Sabotage Connect 4
Connect 4 with a twist! Players 1 and 2 play normally, but players 3 and 4 are spies, who play for the team OPPOSITE of their piece color!

This project has 2 versions. The first is a python script playable in a terminal with `python3 sabotage-connect4.py`
